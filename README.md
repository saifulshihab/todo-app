# todo-app

# Installation

- Client

- [] `cd client`
- [] run `npm install`
- [] run `npm start`

- Server

- [] `cd server`
- [] run `npm install`
- [] run `npm dev`

## API Endpoints

- [ ] Get all todos: /api/todos
- [ ] Create new todo: /api/todos
- [ ] Edit a todo item: /api/todos/:itemId
- [ ] Delete a todo item: /api/todos/:itemId
