# todo-app
Simple todo app using React, Node.js & MongoDB

# Installation

 - Client

- [x] `cd client`
- [x] run `npm install`
- [x] run `npm start`

- Server

- [x] `cd server`
- [x] run `npm install`
- [x] run `npm dev`

## API Endpoints

- [ ] Get all todos: /api/todos
- [ ] Create new todo: /api/todos
- [ ] Edit a todo item: /api/todos/:itemId
- [ ] Delete a todo item: /api/todos/:itemId
